# Manual Test Checklist

- Open `index.html` directly from the filesystem.
- Enter Traditional Chinese text and generate a brief.
- Generate both Suno and Lyria prompt drafts.
- Refresh the page and verify local data restoration.
- Add at least two generation records and verify best-version comparison.
- Export JSON, clear data, and import the exported JSON.
- Export the brief as Markdown.
- Verify imported HTML-like text is displayed as text, not executed.
- Test at mobile and desktop widths.
